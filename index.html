Добро пожаловать в UI Kit!

Используйте меню слева чтобы просмотреть категории готовых компонентов.

📝 Инструкция по использованию UI Kit

Этот UI Kit создан для быстрого прототипирования и внедрения готовых, протестированных элементов на ваши лендинги и веб-страницы. Следуйте этим шагам для эффективной работы.

Шаг 1: Изучите каталог

Перейдите по ссылкам в меню слева (или боковом меню), чтобы просмотреть все доступные категории: Кнопки, Формы, Попапы и т.д.

Шаг 2: Выберите компоненты

В каждой категории вы найдете несколько вариантов (например, `Callback-01`, `Callback-02`). Выберите тот, который лучше всего соответствует дизайну или функциональности вашей задачи.

Важно: Вы можете использовать несколько элементов одновременно (например, `Form-01` + `Callback-02` + `Comments-01`).

Шаг 3: Создание задачи в Trello

Для постановки задачи разработчику используйте название компонента, который вы выбрали. Это гарантирует, что разработчик найдет именно тот код, который нужен.
Также можно кастомизировать элементы (или настраивать элементы), например: менять цвет, размер шрифта, переводить текст

Пример формулировки в Trello-задаче:

"Нужно добавить на главную страницу:
     - Обратный звонок: (Callback-03)
     - Кнопка: (Button-01) 
     - Форму заказа: (Form-04)"

Шаг 4: Внедрение кода (Для разработчика)

Разработчик, используя название блока из задачи, переходит на соответствующую демо-страницу, копирует HTML-код, и подключает CSS и JavaScript из файлов проекта.
